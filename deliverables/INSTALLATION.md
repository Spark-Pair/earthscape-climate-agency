# Installation Instructions

**Prerequisites**
- Python 3.10 or newer
- Git

**Setup**
1. Clone repository.
2. Create virtual environment.
3. Install dependencies using `requirements.txt`.

**Run**
- Start app with `streamlit run main.py`.

**Default Admin**
- Username: `admin`
- Password: `admin123`

**Notes**
- SQLite database auto-creates on first run.
