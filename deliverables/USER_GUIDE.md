# User Guide

**Login**
1. Open the app.
2. Enter username and password.
3. Click `Sign In`.

**Admin Features**
- Users: Add analysts and activate/deactivate accounts.
- Datasets: Upload CSV, validate, clean, save, and assign access.
- Analytics: Use dashboard, prediction, reports, feedback, and performance tabs.
- Access Matrix: Review which analysts can access which datasets.

**Analyst Features**
- Open assigned datasets only.
- Use analytics dashboard, prediction, and reports.
- Submit feedback.
- View own performance logs.

**Dataset Requirements**
Your CSV must include these columns:
- `Year`
- `Month`
- `Temperature`
- `Rainfall`
- `CO2`
- `Humidity`
- `WindSpeed`

**Dashboard**
- Adjust year range.
- Review KPIs and trend charts.
- Check anomaly detection and disaster risk alerts.
- Save alerts snapshot for reporting.

**Prediction**
- Enter values for Year, Month, Rainfall, CO2, Humidity, WindSpeed.
- Click `Predict Temperature`.

**Reports**
- View summary and aggregated report.
- Download CSV or summary text.
- Review saved alerts snapshots.

**Feedback**
- Select subject and write message.
- Submit to support.

**Logout**
- Use `Logout` from sidebar.
